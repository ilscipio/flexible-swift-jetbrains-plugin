# Flexible Swift

[Flexible Swift](https://plugins.jetbrains.com/plugin/31023-flexible-swift) is a plugin for JetBrains IDEs that empowers you to develop, debug, and deploy Swift and iOS applications with confidence.

This repository exists as a community hub and issue tracker.

## Features

- Swift 5.x syntax highlighting and parsing with semantic annotations
- Plist file support with dedicated syntax highlighting, completion, and documentation
- LLDB debugger via Debug Adapter Protocol with line breakpoints
- Swift run configuration with editor gutter run markers
- iOS Simulator tool window with device management (boot, shutdown, erase, open)
- iOS Simulator run configuration with xcodebuild integration
- Corellium cloud device and Appetize.io cloud simulator run configurations
- Go to declaration, go to implementation, go to symbol, find usages
- On-hover documentation for ~370 Swift standard library symbols
- Code completion: local symbols, dot completion, contextual keywords, SwiftUI modifiers
- Inlay hints for parameter names and types
- 15 live templates (sw: prefix), 3 intention actions, 2 inspections
- Code folding, formatting, brace matching, structure view
- Swift SDK auto-detection and configuration
- Color schemes: Default, Darcula, IntelliJ Light

## Installation

You can install Flexible Swift from the JetBrains Marketplace or from the IDE settings.

- **From the Marketplace:** Go to [Flexible Swift](https://plugins.jetbrains.com/plugin/31023-flexible-swift) and click on the Install button. Then restart your IDE to activate the plugin.
- **From the IDE settings:** Go to File > Settings > Plugins and search for Flexible Swift. Click on the Install button and restart the IDE.

## Usage

To get started with Flexible Swift:

1. Install the plugin and restart your IDE
2. Go to **Settings > Languages & Frameworks > Swift** and configure your Swift SDK path
3. Open or create .swift files and start coding
5. Use **Run > Edit Configurations > + > Swift** to create a run configuration
6. For iOS development, use **Swift iOS > iOS Simulator (Xcode)** run configuration

For the full setup guide, see the [plugin documentation](https://plugins.jetbrains.com/plugin/31023-flexible-swift).
